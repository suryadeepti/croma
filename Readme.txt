Steps to Run:

	1.	Install Dependencies: Ensure Python, TensorFlow, Pandas, NumPy, and Flask are installed.
	2.	Generate Data: Run the synthetic data generation script.
	3.	Preprocess Data: Execute the data preprocessing code.
	4.	Train GAN: Train the GAN model using the training script.
	5.	Generate Demand: Run the demand generation code to produce scenarios.
	6.	Deploy API: Start the Flask app to access the model via an API.
